# 🏥 Health Prediction Application

A simple Flask-based web application that collects patient health data and uses AI (Groq LLaMA model) to predict possible health risks based on blood test results.

---

## 🚀 Features

- Add patient details
- Edit patient records
- Delete records
- View all patients
- AI-powered health prediction (Groq API)
- Input validation (email, DOB, numeric checks)

---

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML / CSS
- Groq AI API
